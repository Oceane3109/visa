# 💻 Sprint 4 : Application frontend + suivi des demandes

## 📌 Objectif

Créer une application séparée (Vue.js / React / Angular) consommant les API Spring Boot.

---

## 🖥️ 1. Recherche par numéro de passeport

- L'utilisateur entre un numéro de passeport
- Le système affiche :
  - Liste des demandes associées
  - Statut actuel
  - Tri chronologique

- Possibilité :
  - Créer une demande
  - Associer une demande existante

---

## 📄 2. Recherche par numéro de demande

- Affiche :
  - Détails complets de la demande
  - Informations du demandeur
  - Statut actuel

---

## 📌 3. Cas duplicata

- Recherche par passeport
- Affichage simplifié :
  - Informations minimales
  - Statut : **APPROUVÉ**
  - Indication : dossier terminé

---

## ➕ 4. Création de demande

- Génération automatique d'un **numéro de demande unique**
- Association au passeport

---

## 📲 5. QR Code

- Généré automatiquement après création
- Contient une URL unique

👉 Permet :
- Suivi complet de la demande
- Accès direct au statut et informations

---

## 🔁 6. Règles d'affichage

- Recherche par passeport → tri chronologique
- Affichage clair des statuts
- Historique complet des demandes

---

## 🎯 Résultat Sprint 4

- Application frontend moderne
- Suivi en temps réel des demandes
- QR code pour traçabilité
- Interface utilisateur simplifiée
